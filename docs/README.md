# 文档导航

本目录用于维护 `rshell` 的长期工程文档。

- `DEVELOPMENT.md`：本地开发、调试、测试与发布流程
- `ARCHITECTURE.md`：系统架构、模块职责、关键数据流
- `API_REFERENCE.md`：后端命令与接口约定
- `USER_GUIDE.md`：用户功能与操作说明
- `UX_OPTIMIZATION.md`：用户体验分步优化计划（路线图与勾选清单）

建议：

- 代码目录调整后，优先同步 `DEVELOPMENT.md` 与 `ARCHITECTURE.md`
- 接口字段变更后，及时更新 `API_REFERENCE.md`
