# rshell 用户使用手册

本手册面向最终用户，覆盖从会话管理、终端交互到 SFTP 下载和主机监控的完整使用流程。

## 安装与升级（Windows）

- **推荐**：在 [GitHub Releases](https://github.com/touchfish1/rshell/releases) 任选其一：
  - 直接下载 **`*_x64-setup.exe`（NSIS 安装程序）** 或 **`.msi`**，按向导安装；日常用「开始」菜单里的 **rshell** 启动。
  - 或下载 **`rshell-windows-x64.zip`**：解压后**只会看到安装程序**（`*-setup.exe`），运行它完成安装即可，**不要**指望 zip 里再带一个可直接运行的裸 `rshell.exe`。
- **应用内「在线升级」**只更新**已安装**的那一份，与安装路径绑定。
- 若本机曾把旧版解压到多处，请删掉旧目录，统一用安装后的快捷方式启动，避免误点旧路径里的文件。

## 1. 界面总览

应用分为两个核心页面：

- 首页（主机管理）
  - 主机列表
  - 新增/编辑/删除主机
  - 连通性状态展示
  - 顶栏外观：深色 / 浅色 / 跟随系统（与终端页的终端字体预设一并持久化）
- Terminal 页面（操作页面）
  - 顶部 Session 标签栏
  - 左侧主机列表
  - 中间终端区
  - 右侧主机监控 + SFTP 文件列表

## 2. 主机管理（首页）

### 2.1 新增主机

1. 点击新增按钮进入新增主机弹窗。
2. 填写以下字段：
   - 名称（显示名）
   - 协议（SSH/Telnet）
   - 主机（IP 或域名）
   - 端口（SSH 默认 22，Telnet 默认 23）
   - 用户名
   - 编码（默认 `utf-8`，可改 `gbk`）
   - Keepalive 秒数
   - 密码（可选，若不填连接时再输入）
3. 点击“测试连接”验证可达性。
4. 保存后主机会进入列表。

### 2.2 编辑主机

1. 点击主机行的“编辑”或终端页主机右键“修改主机信息”。
2. 可修改所有连接参数。
3. 密码默认掩码显示，点击眼睛图标后可查看真实密码。
4. 可先“测试连接”再保存，降低配置错误概率。

### 2.3 删除主机

点击“删除”后会从会话列表移除，同时移除对应保存的密码。

### 2.4 在线状态

- 启动后立即探测所有主机可达性。
- 后续按固定周期自动探测。
- 状态用于快速筛选可连接目标，但不等同于 SSH 认证成功。

### 2.5 主机列表键盘（首页）

在主机表格区域按 **Tab** 聚焦列表后，可使用 **↑/↓** 或 **j/k** 移动选中项，**Home/End** 跳到首尾，**Enter** 连接当前选中主机，**F2** 打开编辑弹窗。

## 3. Terminal 页面操作

顶栏提供「快捷键」按钮，打开后可查看当前版本支持的全部快捷键说明。焦点不在中间终端区域时，还可按 `Ctrl+/` 打开说明；说明打开时再按 `Ctrl+/` 或 `Esc` 可关闭。

## 3.1 左侧主机列表交互

- 单击：仅选中高亮
- 双击：打开新 Session 标签（同主机可多标签）
- 右键：打开主机操作菜单（编辑、刷新监控等）
- 键盘：与首页相同的列表快捷键（先 **Tab** 聚焦列表区域）

## 3.2 顶部 Session 标签栏

- **键盘**：`Ctrl+Tab` / `Ctrl+Shift+Tab` 在会话标签间循环切换；`Ctrl+Shift+W` 关闭当前标签。焦点在终端（xterm）内时，`Ctrl+W` 保留给 Shell；焦点在侧栏、SFTP 等区域时可用 `Ctrl+W` 关闭当前标签。将鼠标悬停在终端工作区空白处可看到与字号相关的完整提示。
- 左键标签：切换会话
- 关闭按钮：关闭当前标签
- 右键菜单：
  - 复制 session
  - 关闭右边
  - 关闭左边
  - 关闭其他
- 菜单项会根据上下文自动禁用（如“关闭右边”在最右标签上不可用）

## 3.3 终端交互区

- 支持实时输入输出
- 支持窗口自适应尺寸
- 支持会话级编码解码（UTF-8/GBK）
- 切换标签时自动重同步终端布局，降低光标错位/换行偏移

### 3.4 外观与终端字体

终端页顶栏提供与首页一致的主题切换，并额外提供**终端等宽字体**三种预设（与字号同样保存在本机）。终端配色会随亮/暗主题自动调整。

## 4. SFTP 文件列表

### 4.1 布局说明

右侧区域分为两块：

- 上方固定：主机监控
- 下方：SFTP 文件列表（可滚动）

### 4.2 浏览目录

- 点击目录项进入目录
- 使用“上级”按钮返回父目录
- 根目录下“上级”按钮会自动禁用
- 文件与目录使用不同图标显示（`📄`/`📁`）
- 工具栏路径为**面包屑**：可点击任一路径段跳转到对应目录

### 4.3 下载文件

1. 在文件行（非目录）右键。
2. 选择“下载文件”。
3. 下载任务状态会在页面右下角弹出提示。
4. 下载完成后可点击图标直接打开所在目录。

下载默认保存路径：

- `<系统下载目录>/rshell`

如果同名文件已存在，会自动追加序号避免覆盖。

### 4.4 上传文件（拖拽/选择）

- 点击 SFTP 工具栏的「上传」按钮选择本地文件上传到**当前目录**。
- 或将本地文件直接**拖拽到文件列表区域**松开即可上传到当前目录。
- 为避免界面卡顿，当前版本对单文件大小做了限制（约 8MB）；超出会提示文件过大。

### 4.5 下载失败与重试

右下角下载提示在失败时会显示 **重试** 与 **关闭**；重试会再次发起同一路径的下载。成功任务仍会在数秒后自动收起。

## 5. 主机监控

监控区位于 SFTP 区域上方，显示当前激活会话对应主机指标：

- CPU 使用率
- 内存使用（已用/总量）
- 磁盘使用（已用/总量）
- 最近更新时间

更新机制：

- 自动轮询（固定间隔）
- 手动刷新按钮

说明：

- 仅 SSH 会话支持指标采集。
- 指标采集失败时会在监控区显示错误信息。

## 6. 退出应用与窗口

- 主窗口位置与大小由 Tauri **window-state** 插件在退出时自动保存、下次启动恢复。
- 点击窗口关闭按钮时，应用会**隐藏到系统托盘**并保持现有连接，不会立即退出。
- 系统托盘菜单提供「显示主窗口」与「退出」：
  - 选择「显示主窗口」可恢复并聚焦主窗口；
  - 选择「退出」时，若仍有已连接会话会弹出确认框，确认后断开所有连接并退出；无连接时直接退出。

## 7. 常见问题

### 7.1 SFTP 握手偶发失败

现象：

- `SFTP 列表读取失败: handshake failed: [Session(-8)] Unable to exchange encryption keys`

说明：

- 后端已增加自动重试机制。
- 若仍出现，常见原因是服务端算法策略不兼容或网络中间设备干扰。

### 7.2 终端中文乱码

- 在主机配置将编码设置为正确值（常见 `utf-8`/`gbk`）。

### 7.3 标签关闭后偶发 inactive session

- 多发生于关闭瞬间和轮询并发时，属于竞争窗口问题，前端已做降噪处理。

## 8. 使用建议

- 首次连接先用“测试连接”检查端口可达。
- 生产主机建议按环境命名（如 `prod-web-01`）并使用专门账号。
- SFTP 下载大文件前，先确认本地磁盘剩余空间。
- 若频繁切换主机，建议使用“复制 session”保留历史输出上下文。
